# Django Discord

Aplikacja webowa inspirowana Discordem - komunikacja w czasie rzeczywistym, kanaly, wiadomosci prywatne, role i moderacja.

## Uruchomienie

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Aplikacja dostepna pod adresem: http://127.0.0.1:8000/
